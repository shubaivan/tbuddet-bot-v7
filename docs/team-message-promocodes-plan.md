👤 ЯК ПРАЦЮВАТИМЕ ДЛЯ КЛІЄНТА

При оформленні замовлення з'являється поле "Промокод". Клієнт вводить код → бачить нову суму зі знижкою → оплачує.

Працює і на сайті, і в Telegram-боті, і для гостьових покупок без реєстрації.

━━━━━━━━━━━━━━━━━━━━

🎯 ДВА ТИПИ ПРОМОКОДІВ

1️⃣ МАРКЕТИНГОВІ
   Один код для всіх (наприклад, SPRING20). Можна використати багато разів. Для акцій, розсилок, соцмереж.

2️⃣ ОДНОРАЗОВІ
   Унікальний код, прив'язаний до конкретного клієнта. Спрацює тільки в нього. Наприклад: вибачитися за затримку, подякувати постійному клієнту, повернути лояльність.

━━━━━━━━━━━━━━━━━━━━

⚙️ ЩО НАЛАШТОВУЄ АДМІН

— Тип знижки: відсоток (10%) або фіксована сума (500 грн)
— Дату дії (від—до)
— Мінімальну суму замовлення
— Скільки разів можна використати загалом і одним клієнтом
— Швидкий перемикач "увімк./вимк."

━━━━━━━━━━━━━━━━━━━━

📩 АВТОДОСТАВКА ОДНОРАЗОВИХ КОДІВ

Коли адмін створює персональний код для конкретного клієнта — система сама надсилає його:

— через Telegram (якщо клієнт є в нашому боті);
— або на email (якщо немає).

Адмін у списку бачить статус: "✓ Надіслано через TG", "Очікує", тощо.

🛡 ЗАХИСТ ВІД ЗЛОВЖИВАНЬ

— Один промокод = одне замовлення (повторне натискання не дублюватиме знижку)
— Не можна застосувати два промокоди в одному замовленні (галузевий стандарт)
— Код, прив'язаний до конкретного клієнта, у іншого не спрацює
